1. Ringkasan Problem Statement & HMW

Problem Statement:

Kehadiran akademik (di kuliah) saat ini dilakukan secara manual sehingga rawan kecurangan seperti "titip absen", dan proses rekapitulasi membutuhkan waktu lama karena harus dilakukan secara manual.

How Might We (HMW):

  • Bagaimana membantu bagian akademik memantau dan merekap kehadiran secara real-time?
  • Bagaimana menciptakan sistem yang bisa meminimalisir Human error dalam rekap data?
  • Bagaimana memastikan integritas data dan mencegah praktik "titip absen"?

2. Brainstorming Ide

Tujuan: Semua ide solusi yang muncul (minimal 5)

No. Ide Solusi Deskripsi Singkat
1. Dashboard web admin Menyediakan Dashboard untuk bagian akademik memantau, merekap, dan mengunduh data.
2. Sistem anti "titip absen" Menggunakan QR Code/fingerprint yang berubah berkala.
3. Rekapitulasi otomatis Sistem otomatis merekap data untuk mencegah human error.
4. Laporan data untuk evaluasi Menyediakan data akurat untuk pengambilan keputusan dan evaluasi.
5. Aplikasi mobile Untuk mahasiswa melakukan absen dan untuk dosen me-manage QR Code/Presensi.

4. Konsep Solusi Terpilih

Nama Solusi (sementara):

Sistem Presensi Akademik Terintegrasi

Deskripsi Singkat:

  1. Aplikasi Mobile - Mahasiswa melakukan absen dan dosen mengelola QR Code dinamis yang berubah berkala.
  2. Dashboard - Bagian akademik memantau kehadiran real-time, merekap data, dan mengunduh laporan.

Target User:

  • Dosen, Admin Akademik, Mahasiswa.

Manfaat Utama:

  1. Meningkatkan akurasi dan integritas data.
  2. Mempercepat proses administrasi dengan rekapitulasi otomatis.
  3. Menyediakan data untuk pengambilan keputusan dan evaluasi.

Sketsa / Visualisasi (Alur Sederhana):

Mahasiswa → Scan QR Code → Sistem verifikasi & catatan kehadiran → Admin (web) → Dashboard & Laporan

5. Refleksi Kelompok

Apa tantangan saat brainstorming?

  • Kesulitan memfokuskan ide; beberapa ide terlalu canggih/mahal seperti fingerprint di setiap kelas.
  • Memastikan "titip absen" tidak terjadi harus seimbang dengan biaya dan kemudahan implementasi.

Mengapa solusi yang dipilih?

  • Solusi QR + Dashboard dianggap paling efektif mencegah "titip absen" dan meminimalisir human error.
  • Biaya implementasi relatif murah dibandingkan hardware tambahan.
  • Relevan dengan bidang Bisnis Digital & Komputer karena menggunakan teknologi digital.
  • Transisi dari presensi manual ke otomatis yang fleksibel dan efisien.
